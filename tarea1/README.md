## Instalación

Importante: Requiere [Node.js](https://nodejs.org/) para ejecutar.

```bash
git clone https://github.com/isaac-evs/MyPortfolio.git
cd MyPortfolio
```

```bash
npm install
```

```bash
npm run styles
```

Ahora, abre el archivo `index.html` en tu navegador, o Live Server.
